automatizando formularios web

29 Ene 2009

Hace algunos dias atras estuve ayudando a un amigo que necesitaba crear blogs para sus usuarios de manera automatizada. Seleccionamos el lyceum (una variante del wordpress multiuser).

Una buena manera para crear los usuarios era con las mismas funciones que el sistema utiliza para esa funcion. Pero se me complicaba el tema de separar el codigo que necesitaba de las dependencias que tenia. Asi que opte por otra variante, googleando un poco, recorde el modulo WWW::Mechanize y modificando un poco un script que encontre, obtuve este que les presento a continuación.
