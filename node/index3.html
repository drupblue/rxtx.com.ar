Samba

bluetooth sobre ubuntu en una dell vostro 1310

16 Nov 2008

Udev rule for full access to the USB device

Display information about the plugged in usb devices:

osvaldo@vostro:~$ lsusb
Bus 004 Device 004: ID 0fce:d042 Sony Ericsson Mobile Communications AB

Create a custom udev rule using the ID information (idVendor:idProduct) and your username (OWNER="USERNAME")

$ sudo nano /etc/udev/rules.d/40-custom.rules

SUBSYSTEMS=="usb", ATTRS{idVendor}=="0fce", ATTRS{idProduct}=="d042", MODE="0660", OWNER="osvaldo"
Reload the udev rules:

$ sudo udevcontrol reload_rules
