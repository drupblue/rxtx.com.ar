05 Sep 2008

SVN Time-Lapse View

29 Ago 2008

SVN Time-Lapse View es un visor multiplataforma (corre sobre java) que descarga todas las revisiones de un archivo y le permite revisarlas a simple vista moviendo hacia uno u otro lado el control de una linea de tiempo. Mientras mueve el control, podrá ver visualmente las diferencias entre la versión actual y la previa. De esta manera, es fácil ver la evolución de un archivo, y le resulta muy sencillo ver en que momento las líneas de código fueron adicionadas o eliminadas.

mejorando el blog

27 Ago 2008

Le habilite los modulos:

Locale
Para ponerle el idioma español en la interface
Tagadelic
Para crear una nube de tags (tagcloud) con la taxonomía libre que utilizo para clasificar los posts
Xmlsitemap
Aunque es un development snapshot para drupal 6, se lo instale y esta funcionando ok al parecer. Es para exportar el sitemap del site a Google y ayudarlo a indexarlo.
Analytics
Es para utilizar las estadisticas de Google (antiguo urchin)
